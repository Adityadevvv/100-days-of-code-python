# 🐍 100 Days of Python Code Challenge

Welcome to my **100 Days of Python Code** journey! 🚀

This repository documents my progress as I learn Python from the ground up through a structured 100-day coding challenge. My goal is to build a strong foundation in Python programming by writing code every single day and continuously improving my problem-solving skills.

## 🎯 Goal

* Complete all **100 days** of the Python challenge.
* Upload the project or exercise completed each day.
* Build consistency through daily coding.
* Strengthen my understanding of Python fundamentals and real-world programming concepts.

Each day's work is organized into its own folder.


Each folder may contain:

* Python source code (`.py`)
* Additional assets (if required)
